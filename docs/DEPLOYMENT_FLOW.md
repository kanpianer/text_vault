```
┌─────────────┐
```

## 🔄 数据流程

### 1️⃣ 创建保险库

```
用户输入密码
    │
    ▼
浏览器派生密钥 (PBKDF2)
    │
    ├─► AES-256 密钥 (加密用)
    └─► Auth Hash (认证用)
    │
    ▼
加密数据 (AES-GCM)
    │
    ▼
发送到 Worker (/api/vault/create)
    │
    ▼
Worker 验证并存储到 KV
    │
    └─► 存储: auth_hash_double (SHA-256)
    └─► 存储: encrypted_data
    └─► 存储: salt_enc, salt_auth
```

### 2️⃣ 解锁保险库

```
用户输入密码
    │
    ▼
请求 salt (/api/vault/:name/salts)
    │
    ▼
浏览器派生密钥 (使用返回的 salt)
    │
    ▼
发送 auth_hash 到 Worker (/api/vault/get)
    │
    ▼
Worker 验证 SHA-256(auth_hash)
    │
    ├─► 匹配 ✅ 返回 encrypted_data
    └─► 不匹配 ❌ 返回 401 错误
    │
    ▼
浏览器解密数据 (AES-GCM)
    │
    ▼
显示内容
```

### 3️⃣ 保存内容

```
用户编辑内容
    │
    ▼
浏览器加密新数据 (使用内存中的密钥)
    │
    ▼
发送到 Worker (/api/vault/update)
    │  + auth_hash (证明身份)
    │  + encrypted_data
    │
    ▼
Worker 验证 auth_hash
    │
    ▼
更新 KV 中的数据
    │
    ▼
返回成功 ✅
```

## 🔐 安全架构

```
┌─────────────────────────────────────────────────────────┐
│              客户端（浏览器）                               │
│  ┌─────────────────────────────────────────────────┐    │
│  │  原始密码 → PBKDF2(600k) → AES Key + Auth Hash  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  ┌─────────────────────────────────────────────────┐    │
│  │  明文数据 → AES-GCM Encrypt → 加密数据           │    │
│  └─────────────────────────────────────────────────┘    │
└───────────────────┬─────────────────────────────────────┘
                    │
                    │ HTTPS (加密传输)
                    ▼
┌─────────────────────────────────────────────────────────┐
│              服务端（Cloudflare）                          │
│  ┌─────────────────────────────────────────────────┐    │
│  │  存储: SHA-256(Auth Hash)  ◄── 只存储哈希的哈希  │    │
│  │  存储: 加密数据             ◄── 无法解密         │    │
│  │  存储: Salt                ◄── 公开，不敏感     │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘

🔒 服务器永远无法：
   ❌ 看到原始密码
   ❌ 解密用户数据
   ❌ 恢复 AES 密钥
   
✅ 即使服务器被攻击，用户数据依然安全！
```
