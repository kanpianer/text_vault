## 📌 重要提醒

### ⚠️ 部署前必读
- [ ] 确保已运行 `npm install` 和 `npm run build`
- [ ] 准备好 Cloudflare 账户（免费）
- [ ] 预计需要 5-10 分钟完成部署

### ⚠️ 部署时注意
- [ ] KV Binding 名称必须是 `VAULTS`（大写）
- [ ] 修改 `worker.js` 中的 Pages URL
- [ ] 上传 `dist` 文件夹**内**的文件，不是文件夹本身

### ⚠️ 部署后验证
- [ ] 能访问应用界面
- [ ] 能创建保险库
- [ ] 能保存和读取数据
- [ ] 锁定后能重新解锁
